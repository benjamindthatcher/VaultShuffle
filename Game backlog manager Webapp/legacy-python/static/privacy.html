Privacy

This local project version is designed to keep your backlog on your own machine.

Local Data

The current app stores your game library, Steam session, and optional Steam Web API key in a local database file inside the project folder.

Steam Sign-In

Steam sign-in is handled through Steam OpenID. Vault Shuffle receives your SteamID64, not your Steam password.

Steam Artwork

If a game has a Steam AppID, the browser may request artwork from Steam's public image servers to display game covers and headers.

Future Hosting

If this becomes a hosted product later, this page should be replaced with a full privacy policy covering accounts, sync, analytics, and storage.
