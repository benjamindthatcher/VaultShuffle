How It Works

Built around the actual decisions you make before playing, not a fake productivity dashboard.

01

Sign in with Steam

Steam confirms who you are, then Vault Shuffle can import owned games with playtime when your Web API key is added.

02

Filter by the night

Hide completed games, search the library, sort by playtime, and keep the list scrolling inside the app where it belongs.

03

Shuffle the vault

Get a focused pick from your actual backlog, with the Steam cover, progress, notes, and next actions beside it.

Made for big libraries

The game list has its own scroll area, so a 245-game backlog still feels contained inside a normal browser window.

Steam artwork included

Games with Steam AppIDs pull public artwork from Steam's CDN, and new games can be found through Steam store search.

Database backed

The project now uses a local SQLite database, with sessions, Steam users, settings, and games all ready to grow.

Built to choose

The whole point is the moment of decision: less scrolling, less guilt, more actually launching something.

A backlog manager with taste.

Vault Shuffle keeps the dense bits useful, then wraps them in a blue-black glass interface with a purple selected-card accent.
